Wishlink Pixel is your one stop solution to calculate ROI on affiliate marketing spend!

== Description ==

Wishlink(WL) Pixel tracks the purchases from the traffic sent by WL affiliated sources, made on the woocommerce enabled website. All the affiliate transaction information are also sent to Wishlink.

== Installation ==

Upload the Wishlink plugin to your website, activate it, and then enter your unique website name and currency in which you sell your products.
You're done!

== Frequently Asked Questions == 
Q. Do the wishlink Pixel tracks user info?
A. No! We do not track any user info, we only capture the order related information and the interaction of user with site.
